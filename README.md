**Buscador de Moedas - API Banco Central**
*Sobre o Projeto*

Este projeto foi desenvolvido em C# utilizando Windows Forms e consome a API pública do Banco Central do Brasil para buscar a cotação de moedas estrangeiras.

**O sistema permite:**

✅ Consultar moedas

✅ Buscar valores da última semana

✅ Exibir resultados na interface

✅ Gerar relatório Excel automaticamente

✅ Exportar dados das cotações
-------------------------------------------------------------------------------

**Funcionalidades:**

🔎 Consulta de Moedas


**O sistema realiza requisições para a API do Banco Central e retorna:**

Data da cotação
Valor de compra
Valor de venda
Variação cambial

Histórico da Última Semana

**O usuário pode consultar os valores dos últimos dias úteis da moeda selecionada.**

*Exemplo:*

Dólar
Euro
=============================================================================================
**Exportação para Excel**
Após a consulta, o sistema gera automaticamente um arquivo .xlsx contendo:

Datas
Valores
Histórico completo das cotações

Ideal para:

Relatórios
Estudos
Controle financeiro
Comparativos
